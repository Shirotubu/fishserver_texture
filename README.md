# fishserver_texture
Minecraft 1.12.2 server_resoucepack
fishserverのリソースパックのテクスチャ画像。

## fishserverのテクスチャ作成を初めた方に
wikiに詳細があります。その手順に沿ってgithubで開発できる環境にしてください。

## 必須情報
1. 32*32のキャンバスサイズでドットイラストをかく。
（なるべく重複はないように！）
2. .pngで出力する。 ※.jpegだと透過されなかったりゲーム内で適用されない為注意
